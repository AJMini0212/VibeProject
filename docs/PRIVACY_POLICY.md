## 1. 개인정보의 수집

### 수집하는 개인정보 항목

CafeMatch는 다음과 같은 개인정보를 수집합니다:

#### 1-1. 회원가입 시
- 이메일 주소
- 비밀번호 (암호화 저장)
- 표시 이름
- 프로필 사진 (선택사항)
- 자기소개 (선택사항)

#### 1-2. 서비스 이용 중
- 작성한 리뷰 (텍스트, 별점)
- 작성한 댓글
- 좋아요한 리뷰
- 팔로우/팔로워 정보
- 방문한 카페 위치 정보
- 즐겨찾기 목록
- 계정 활동 로그

#### 1-3. 기술적 정보
- 기기 ID (UUID)
- IP 주소
- 앱 버전
- OS 및 기기 정보
- 에러 및 크래시 로그

#### 1-4. 위치 정보 (선택사항)
- GPS 좌표
- 주변 카페 검색 시에만 수집

### 수집 방법
- 회원가입 시 직접 입력
- 서비스 이용 중 자동 수집
- Firebase Analytics를 통한 수집
- Firestore 데이터베이스 기록

---

## 2. 개인정보의 이용 목적

수집한 개인정보는 다음의 목적으로만 이용됩니다:

- **서비스 제공**: 회원 관리, 콘텐츠 제공
- **서비스 개선**: 사용자 경험 개선, 기능 개발
- **통계 분석**: 서비스 이용 통계, 성능 분석
- **보안**: 부정 이용 방지, 서비스 보안
- **고객 지원**: 문의 응답, 기술 지원
- **법적 의무**: 법률에서 요구하는 경우

### 목적 외 이용 금지
- 명시된 목적 이외의 용도로 이용하지 않습니다
- 이용자의 사전 동의 없이 목적을 변경하지 않습니다

## 3. 개인정보의 보호 및 보안

### 보안 조치

#### 3-1. 기술적 보안
- **암호화**: 전송 중 SSL/TLS 암호화
- **저장**: 비밀번호 해싱 저장
- **인증**: Firebase Authentication 사용
- **접근 제어**: Firestore 보안 규칙

#### 3-2. 관리적 보안
- 최소 권한 원칙
- 정기적인 보안 감사
- 접근 로그 기록

#### 3-3. 물리적 보안
- Firebase 클라우드 인프라 사용
- Google 데이터센터 보안

### 비밀번호 관리
- 비밀번호는 일방향 암호화로 저장
- 최소 8자 이상 권장
- 주기적인 변경 권장

### 보안 약점 보고
- 보안 문제 발견 시: security@cafematch.example.com

## 4. 개인정보의 보유 및 이용기간

### 보유 기간

| 정보 | 보유 기간 |
|------|---------|
| 회원가입 정보 | 계정 삭제 시까지 |

- 보유기간 만료 시 자동 삭제
- 사용자 요청 시 즉시 삭제
- 법적 의무가 있는 경우 보관
- 파기 시 복구 불가능한 방식으로 처리

## 5. 개인정보의 제3자 제공

### 원칙
- 이용자의 동의 없이 제3자에게 제공하지 않습니다
- 법적 의무가 있는 경우는 예외

### 예외 사항
- 법령에 따른 요청 (수사 기관 등)
- 서비스 제공에 필수적인 경우
  - Firebase (Google) - 클라우드 호스팅
  - Firebase Analytics - 분석
  - Firebase Crash Reporting - 에러 추적

### 외부 서비스
- **Google Maps API**: 지도 서비스 (위치 정보)
- **Naver Place API**: 카페 정보 (공개 정보)

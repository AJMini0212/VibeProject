🎯Phase별 개요

Phase 1

지도 & 검색 (MVP)

기간: 10-11주차

시간: 14시간

목표: 기본 검색과 브라우징 기능 완성

  • Google Maps 통합
  • Naver Place API 연동
  • 검색 & 필터링 UI
  • 상세 페이지
Phase 2

리뷰 & 인증

기간: 12-13주차

시간: 18시간

목표: 사용자 시스템과 리뷰 커뮤니티 구현

  • Firebase 인증
  • 사용자 프로필 시스템
  • 리뷰 작성/조회
  • 평점 및 이미지 처리
Phase 3

소셜 & 배포

기간: 13-15주차

시간: 20시간+

목표: 커뮤니티 기능 완성 및 앱 배포

  • 팔로우/팔로워 시스템
  • 활동 피드
  • 리뷰 상호작용
  • 배포 & 모니터링

🔗의존성 관계

각 Phase는 순차적으로 진행되며, Phase 1의 70% 이상 완료 후 Phase 2를 시작할 수 있습니다.

  • Phase 1 완료: 지도에서 가게 50개+ 표시, 검색 기능 정상
  • Phase 2 완료: 리뷰 5개+, 계정 생성 및 로그인 정상
  • Phase 3 완료: 팔로우 기능 정상, 피드 실시간 업데이트
  • 배포 준비: 크래시 0, 성능 지표 만족
